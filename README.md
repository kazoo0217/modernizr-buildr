# modernizr-buildr

npm-scripts を使った modernizr をビルドするときに使える

## Usage

modernizr をインストールする

```shell
$ npm install -g modernizr
```

## Configure

edit `src/config.json`

デフォルトでは HTML5 Boilerplate のカスタム設定を使用している

## Build

```shell
$ npm run modernizr
```

## Reference

- [Modernizr](https://modernizr.com)
- [HTML5 Boilerplate](https://html5boilerplate.com)
